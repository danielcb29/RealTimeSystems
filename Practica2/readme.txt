Este directorio contiene la practica 2 del curso STR, el folder barrera contiene el codigo fuente de la solucion del problema en un proyecto de eclipse donde esta tambien el ejecutable para Linux. 

Barrera se puede compilar y ejecutar mediante eclipse o tecleando 

g++ main.cpp barrera.cpp -o main -lpthread 

donde se creara el ejecutable main. 

El folder Signals contiene la solcion al problema 2, se puede ejecutar con el ejecutable para Linix o compilar y ejecutar mediante eclipse o tecleando 

g++ main.cpp -o main -lpthread 

donde se creara el ejecutable main. 

La practica se encuentra disponible tambien el directorio github del curso: https://github.com/danielcb29/RealTimeSystems
